Outil de test de formules

Un outil pour filtrer les lignes d'un document budgétaires

Pour le moment, cet outil ne filtre que les données des Comptes Administratifs du Département de la Gironde pour les années 2016 et 2017... mais rien n'empêcherait de taper dans les autres années ou d'autres collectivités/établissements publics ou les BP, etc.

Exemples de formules :

  • D : toutes les dépenses
  • R : toutes les recettes
  • RI : toutes les recettes d'investissement
  • DF : toutes les dépenses de fonctionnement
  • DF*F4 : toutes les dépenses de fonctionnement de la fonction 4
  • DF*(F4+F5) : toutes les dépenses de fonctionnement des fonctions 4 et 5
  • DF*(N64111+N6451) : toutes les dépenses de fonctionnement des natures 64111 et 6451 (salaire + URSSAF)
  • DF*(N64111+N6451)*F52 : toutes les dépenses de fonctionnement salaires+URSSAF lié à la fonction 52 (Personnes handicapées)
  • RI*C16 : toutes les recettes d'investissement du chapitre 16 (emprunts)
